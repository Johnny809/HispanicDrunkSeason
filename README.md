# DealVista SaaS Landing Page

## Quick Start
```bash
npm install
npm run dev
```

A premium, responsive Next.js landing website for **DealVista**, an AI shopping assistant.

## Tech Stack
- Next.js (App Router) + TypeScript
- Tailwind CSS
- Framer Motion animations
- lucide-react icons

## Pages
- `/` landing page
- `/pricing` pricing + FAQ
- `/demo` interactive AI-style recommendation demo
- `/privacy` privacy policy
- `/terms` terms of service

## Highlights
- Sticky header and premium hero layout with CTA pair
- Social proof, features grid, how-it-works flow, testimonials, and pricing teaser
- Motion with reduced-motion accessibility support
- Local mock recommendation engine on demo page (no external API)
